# SEOCat
A bot for scan and automatize SEO

### Build
1. Clone  
2. Install dependencies `npm i`  
3. Start server `npm start`

### Features
- Metatag check.  
- Keywords and relativity.  
- H1 and content length check.  
- Title and description optimization.  
- Performance optimization.  
- Friendly UI.  
- Proxy Pipe for scan websites without exposes the user.
